Title: Appointment Scheduling Application
Purpose: To allow a user to add, delete, and modify customers and appointments within an application.
Author: Jared Hines
Contact info:
	Email - jhine85@wgu.edu
Student Application Version:0.1
Date: 7/27/2022
IDE: IntelliJ IDEA Community Edition 2021.1.3 x64
JDK: Java SE 17.0.1
SDK: JavaFX-SDK-11.0.2

Directions:

Start by opening IntelliJ IDEA. Open the application code within the program and make sure you have all
necessary files downloaded to run the program. Run the application by clicking the green button in the 
top right corner. Enter the username and password. A dashboard will appear with all of the 
appointment and customer information. From there, you can press a button to add, modify, or delete customers
and appointments. For a full view of customer and appointment info, you can select 'Print' from the menu bar,
then select 'Reports' and a window will pop up with more information if needed. To close the application, press the 'X'
button in the top right corner or select 'File' then select 'close'.

Part A3F: To view additional reports, select 'Print' from the menu bar at the top of the page. This will bring you to
the page where you can view customer appointment by type and month. It will also show the schedule for the contact
information. For the the additional reports, the total amount of hours for the contact is shown. Also, the total number
of login attempts for the application is shown and reported in the top right corner, which is different from the
login_activity.txt file.

MySQL: mysql-connector-java-8.0.25
